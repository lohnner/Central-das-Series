# Livros Loner

Acervo de livros em edições brasileiras e em português:

https://lohnner.github.io/Livros-Loner/

## Páginas

- `index.html` — página inicial
- `livros/index.html` — catálogo com 20 livros, busca, alfabeto e filtros
- `livros/livro.html` — página dinâmica de detalhes e progresso de leitura
- `login.html` — acesso à conta
- `perfil.html` — estante e progresso do leitor

Os dados dos livros ficam em `assets/js/books.js`. O projeto utiliza HTML, CSS e JavaScript sem dependências de build.
